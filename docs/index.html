개인정보 처리방침

본 앱은 사용자의 개인정보를 소중하게 생각하며, 아래와 같은 방침에 따라 개인정보를 수집, 이용, 보관합니다.

1. 수집하는 개인정보 항목

- 소셜 로그인 플랫폼(카카오, 구글)을 통해 제공받은 사용자 정보 (예: 이름, 이메일, 프로필 사진 등)

- 생성한 콘텐츠 (예: 동화 내용, 음성, 이미지 등)

2. 개인정보 수집 방법

- 카카오, 구글 소셜 로그인 시 플랫폼을 통해 자동 제공

- 앱 내 콘텐츠 생성 시 사용자 입력 또는 생성 정보 수집

3. 개인정보의 이용 목적

- 사용자 인증 및 로그인 기능 제공

- 사용자 맞춤형 AI 콘텐츠 생성 및 저장

- 사용자별 콘텐츠 이력 및 서비스 기록 관리

4. 보유 및 이용 기간

- 회원 탈퇴 또는 서비스 종료 시까지

- 관련 법령에 따라 보관이 필요한 경우 해당 기간 동안 보관

5. 개인정보 제3자 제공

- OpenAI API: 콘텐츠 생성을 위한 입력 정보 전달

- Amazon Web Services: 데이터 저장 및 보안 처리

- 제3자 제공은 최소한의 범위로만 하며, 명시된 목적 외로는 사용하지 않습니다.

6. 개인정보 처리 위탁

- Amazon S3: 이미지 및 음성 데이터 저장

- AWS RDS (PostgreSQL): 사용자 데이터 및 콘텐츠 저장

7. 이용자의 권리

- 소셜 로그인 플랫폼 설정을 통해 개인정보 제공 내역 확인 가능

- 앱 내 고객센터 또는 이메일을 통해 탈퇴 및 정보 삭제 요청 가능

- 관련 문의는 앱 내 고객센터 또는 이메일을 통해 가능합니다.

8. 개인정보 보호를 위한 조치

- 민감 정보 암호화 및 접근 제한

- 데이터 접근 로그 관리 및 보안 정책 시행
